This repository contains code to support a paper on an Agentic AI Privacy Hub The code contains the following

1. negotiate_retention.py -> A Python script that emulates the negotiation between two AI agents to set data retention

2. negotiate_retention_sample_output -> A sample output

3. user_preferences.py -> A Python script showing how verbose user's privacy preferences can be interpreted and enforced by an AI agent

4. user_preferences_sample_output.py -> A sample output

5. Prompts -> The prompts and responses from leading LLMs, referenced in the paper.
